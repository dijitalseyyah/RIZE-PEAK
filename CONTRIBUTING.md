# Katkı Sağlama Rehberi

RIZE-PEAK projesine katkıda bulunmak istediğiniz için teşekkürler! Rize'nin tarihini ve kültürünü belgelemek adına her türlü katkı değerlidir.

## Nasıl Katkı Sağlayabilirsiniz?

1.  **Hata Raporlama:** Bir hata veya eksik bilgi bulursanız lütfen bir issue açın.
2.  **Yeni Bilgi Ekleme:** Rize ile ilgili tarihi belgeler, fotoğraflar veya analizler eklemek için Pull Request gönderebilirsiniz.
3.  **Düzeltmeler:** İmla hataları veya yanlış bilgileri düzeltebilirsiniz.

## Süreç

1.  Projeyi forklayın.
2.  Anlamlı bir branch ismiyle yeni bir branch oluşturun (`feature/yeni-bilgi` gibi).
3.  Değişikliklerinizi yapın.
4.  Commit mesajlarınızın açıklayıcı olmasına özen gösterin.
5.  Pull request gönderin.

Katkılarınız için şimdiden teşekkürler!
